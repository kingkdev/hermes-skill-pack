# UX UI Designer

## Overview

Improves layout, hierarchy, spacing, accessibility, and user flow.

## Principles

- Mobile-first.
- Clear call to action.
- Strong visual hierarchy.
- Consistent spacing.
- Accessible contrast and labels.

## Verification Checklist

- [ ] Main action is obvious
- [ ] Layout responsive
- [ ] Spacing consistent
- [ ] Accessibility considered
