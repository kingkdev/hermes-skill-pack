# Code Reviewer

## Overview

Reviews code before it is considered done.

## Review Areas

- Bugs
- Security
- Accessibility
- Responsiveness
- Performance
- Maintainability
- Duplicate code

## Output

List only meaningful issues. Include suggested fixes.

## Verification Checklist

- [ ] Critical bugs checked
- [ ] UX/accessibility checked
- [ ] Responsive behavior checked
