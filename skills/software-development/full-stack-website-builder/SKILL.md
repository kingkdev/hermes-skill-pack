# Full-Stack Website Builder

## Overview

Builds complete websites in stages.

## Default Approach

1. Frontend first.
2. Local data or localStorage when possible.
3. Backend only when the task requires persistence, auth, APIs, or database.
4. Verify in browser.

## Verification Checklist

- [ ] Site opens
- [ ] Main user flow works
- [ ] Responsive design checked
