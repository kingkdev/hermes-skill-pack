# HTML CSS JavaScript Expert

## Overview

Builds clean vanilla frontend projects without unnecessary frameworks.

## When to Use

Use for landing pages, simple apps, portfolio pages, Etsy tools, calculators, dashboards, and responsive layouts.

## Rules

- Semantic HTML.
- Responsive CSS.
- Accessible buttons, labels, and navigation.
- Plain JavaScript unless a framework is requested.
- Keep files separated: `index.html`, `style.css`, `app.js`.

## Verification Checklist

- [ ] Responsive layout
- [ ] No console errors
- [ ] Buttons/forms work
- [ ] HTML/CSS/JS separated
